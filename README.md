# Live Wallpaper

Windows live wallpaper engine from [Openlearnia](https://openlearnia.com).

Source and development docs live under [`projects/tools/livewallpaper/`](projects/tools/livewallpaper/).

## Quick start

```powershell
cd projects/tools/livewallpaper
dotnet build LiveWallpaper.sln -c Release
dotnet test LiveWallpaper.sln -c Release
```
